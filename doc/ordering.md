# Ordering

## Ordering of data structures

I also use this sometimes to order functions by their return value.

* solution
  * `solver::Solution`
* outcome
  * `solver::Outcome`
* game state
  * `data::GameState`
* game
  * `data::Game`
* board
  * `data::Board`
* sub-board
  * `data::SBoard`
* rows *
  * `[data::Row; 3]`
* row
  * `data::Row`
* board play
  * `data::Play`
* sub-board play
  * `data::SPlay`
* board location
  * `data::Loc`
* sub-board location
  * `data::SLoc`
* slots *
  * `[data::Slot; 9]`
  * `[data::Slot; 3]`
* slot
  * `data::Slot`
* board indexes
  * `data::RI`
  * `data::CI`
  * `data::BI`
* sub-board indexes
  * `data::SRI`
  * `data::SCI`
  * `data::SBI`
* player
  * `data::Player`
* u8 *
* bool *
